Hyper Text Markup Language

The Hyper Text Markup Lanuage or HTML is a markup language that is used in order to create websites or web applications. HTML is used to define the web page content and its layout. HTML was created along with the WWWtogether which was based on Standard Generalized Markup Lanugage or SGML as its publishing language.

XHTML

XHTML was based on HTML 4.0 that follows the Extensible Markup Lanugage (XML) rules which means XHTML can create cerain set of rules of any kind of data that can be used for describing or creating the web page layout.

HTML 4.0 vs HTML 5 vs XHTML

HTML 4 is the oldest version of HTML that is still used. HTML 4 has a loose syntax since having a closing tag was optional. HTML 4 is great for new developers since it is very easy to use and its very easy to adapt. HTML 4 has a disadvantage since loose syntax isnt great if you want to have an specific outcome for different browsers. XHTML was created to fix the problem of HTML 4, which was having a loose syntax. In XHTML all opening tags must have a closing tags. Since XHTML is more restricted, it wont allow you to have mistakes.

HTML 5 is a new version of HTML which is still developing. HTML 5 is becoming more popular these days because of its new features. HTML 5 has also a loose syntax but it isnt a problem now since some programs that is used to create HTML auto generates a closing tags. HTML 5 still uses some of the best features of HTML 4 but it also have some new tags. One of the popular feature of HTML 5 is that you can embed videos directly without the use of plugins.

Document Structure

HTML 4 needs to be started by a <html> tag and needs to end with </html> tag. All other elements should appear inside <html> and </html>. The head element is the one that appears first after the <html> tag. Inside the head, the elements title, meta data, and script is placed together with the link of external css and some other resources that is needed. The contents of the web pages is placed inside the body elemnt. All of these are the basic structure of HTML document structure, since the HTML 5 has support from modern browsers, it is more ideal to use the new elemts to make the web pages structure more creative.

Elements vs Tags

  • Elements are composed of a start tag(i.e <p>) a content and a closing tag(i.e. </p>), the following is an example: <p> example </p>.
  • Tags are composed of a two angle brackets and in the middle of those two angle brackets is an element name with or without attributes.

Attributes

Attributes in HTML provides extra information about an element for example the href attribute. The <a> tags defines the HTML links and the address of the link is defined by href(i.e. <a href = “www.example.com” >) Link to www.example.com.

Examples:

  • alt: When the image can't be displayed, it specifies a replacement text for an image.
  • disabled: Specify an input element to be disabled
  • href: Defines the web address for a link
  • src: Defines the web address for an image

Global Attributes

Global attributes are attributes that are common to all elements in HTML, which means they can be used by all other elements.

Examples:

  • accesskey: Specify a key to activate a certain element
  • class: Lists of classification names which an elemnt belongs
  • contenteditable: Specify if a content of an element is editable
  • contextmenu: Specifies the context menu of a certain element
  • tabindex: Specifies if an element is focusable and its order the sequence of focusable elements
  • draggable: Specify if an element is draggable
  • hidden: Specify if an element is no longer relevant

Custom Non-Visible Data Attributes

Custom nonvisible data attributes are attributes whose name starts with data- for the purpose of storing custom data to the web page or web application.

Other Concepts

MATH ML, SVG

Mathematical Markup Language (MathML) purpose is to facilitate the use of mathematical content on the web page or the web applications. Scalable Vector Graphics are used to define vector graphics on the web in XHTML format.

WAI-ARIA

WAI-ARIA or the Web Accessibility Initiative Accessible Rich Internet Applications purpose is to make web page contents or web applications accessible to the person with disabilities.



Character References

Character References in HTML are used to refer special characters which are not normally seen in the keyboard.

HTML Preproccessors

Preporocessors are the process of transforming a data to another data. HTML preprocessors goal is to make it easier to encode html codes. Some examples of it are the Haml also known as HTML abstraction markup language.


Bit Map vs Vector Graphics

Vector Graphics uses mathematical formulas in order to represent an image while Bit Mapped Graphic uses pixels in order to represent an image. The advantage of Vector Graphics is that even if you try to zoom in the image, the quality wont change compared to Bit Mapped Graphics, if you try to zoom in an image, it becomes pixalated. The disadvantage of Vector Graphics is that if you want to create a complex image, the formulas used is also more complex while in Bit Mapped Graphics, you dont need to solve any formulas since you only need to store the image inside the bits.
